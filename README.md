# bnsfinder — test repository

## Tests

| Directory                            | Purpose                                                      |
| ------------------------------------ | ------------------------------------------------------------ |
| `tests/ripple_differential_fit/`     | Differential fitting of ripple waveforms to H1/L1 strain.    |

More tests will be added as new subdirectories.
